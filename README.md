# Student Manager Application

## 1. Mô tả dự án
Ứng dụng quản lý sinh viên cơ bản sử dụng C# WinForms.
Hỗ trợ các chức năng: Thêm, Sửa, Xóa, Tìm kiếm và lưu trữ dữ liệu ra file.

## 2. Hướng dẫn chạy
1. Clone repo về máy: `git clone <https://github.com/phuoctnm/TranNhatMinhPhuoc_52422_StudentManager.git>`
2. Mở file `StudentManagerApplication.sln` bằng Visual Studio 2022.
3. Nhấn Start (F5) để chạy chương trình.

## 3. Danh sách 5 Commit chính
1. Initial commit: Khởi tạo dự án.
2. UI: add main form layout and controls.
3. Model: add Student class.
4. Feature: implement add/update/delete.
5. Feature: add search function.
